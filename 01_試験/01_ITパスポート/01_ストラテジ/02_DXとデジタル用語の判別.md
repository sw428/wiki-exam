# DXとデジタル用語の判別

## 目的

このファイルは、ITパスポートで混同しやすい「デジタル〇〇」の用語を、問題文の手がかり語から切り分けるための整理です。

特に、デジタルサイネージ、デジタルディバイド、デジタルトランスフォーメーション、デジタルネイティブを見分けます。

## 問題文の要約

「ITやAIを使って、会社のやり方や商売の形を大きく変えることを何というか」を聞いています。

単なるパソコン導入や業務の一部効率化ではなく、仕事やビジネスそのものを変える話です。

## 出題者の意図

出題者が見たいのは、「デジタル」が付く似た用語を、後ろの言葉で見分けられるかです。

見るべきなのは `デジタル` ではなく、その後ろの言葉です。

```text
signage = 看板
divide = 分断、格差
transformation = 変革
native = 生まれつき、その環境で育った人
```

## 用語の意味レイヤー

### デジタルトランスフォーメーション

`digital` は、デジタル技術やITを表します。

`transformation` は、変形、変化、変革を表します。

つまり、デジタルトランスフォーメーションは「デジタルによる変革」です。

ただし試験では、単なるIT化ではありません。

## IT化とDXの違い

### 普通のIT化

- 紙の書類をExcelにする
- 手作業をシステムで楽にする
- 予約をWebで受け付ける

これは、主に効率化です。

### DX

ITやAIを使って、次のような変化まで起こす場合はデジタルトランスフォーメーションです。

- ビジネスモデルを変える
- 新しいサービスを作る
- 会社の仕組みを変える
- 顧客への価値を変える
- 新たな付加価値を生み出す

## 一段深い判別ポイント

| 手がかり語 | 用語 | 住み分け |
| --- | --- | --- |
| 看板 / 広告 / 表示装置 / ディスプレイ | デジタルサイネージ | 電子看板 |
| 格差 / 使える人と使えない人 / 情報弱者 / 地域差 | デジタルディバイド | 情報格差 |
| ビジネスモデル刷新 / 新しい価値 / 会社の仕組みを変える | デジタルトランスフォーメーション | デジタルによる変革 |
| 世代 / 若者 / 生まれた時からネットに慣れている | デジタルネイティブ | デジタル環境で育った世代 |

## 問題の核心

線を引くなら、次の語です。

- ITやAIを活用
- ビジネスモデルの刷新
- 新たな付加価値を生み出す

特に強い手がかりは「ビジネスモデルの刷新」です。

これが出たら、デジタルトランスフォーメーションを第一候補にします。

## 正解

デジタルトランスフォーメーション。

## 他の選択肢を切る理由

ア: デジタルサイネージは電子看板です。見分け語は「看板」「広告表示」「ディスプレイ」です。問題文は広告表示ではなく、ビジネスモデルの変革を言っているため切ります。

イ: デジタルディバイドは情報格差です。見分け語は「格差」「高齢者」「地域差」「使える人と使えない人」です。問題文に格差の話がないため切ります。

ウ: デジタルトランスフォーメーションはITによる変革です。見分け語は「刷新」「変革」「付加価値」「ビジネスモデル」です。問題文そのものがこれです。

エ: デジタルネイティブは、生まれた時からデジタル環境に慣れている世代です。見分け語は「世代」「若者」「生まれた時から」です。問題文は人の世代ではなく、企業活動の変革なので切ります。

## 一瞬暗記フレーズ

```text
ビジネスモデル刷新 = DX
看板 = サイネージ
格差 = ディバイド
変革 = トランスフォーメーション
生まれつき世代 = ネイティブ
```

## 記憶固定イメージ

コンビニで考えると分かりやすいです。

レジを電子化するだけなら、主にIT化です。

AIで売れ筋を予測し、アプリで個人ごとにクーポンを出し、無人レジにして、店舗運営や顧客価値そのものを変えるならDXです。

道具をデジタルにするだけではなく、商売のやり方ごと変えるのがDXです。

## 最後に一言

「刷新」「付加価値」「ビジネスモデル」が出たら、デジタルトランスフォーメーション。

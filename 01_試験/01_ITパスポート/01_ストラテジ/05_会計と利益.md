# 会計と利益

## 目的

このファイルは、ITパスポートの会計・利益計算問題を、どの収益や費用がどの利益に効くかで判断できる形に整理する場所です。

営業利益、経常利益、損益分岐点、流動比率、ROE、営業外収益、販売費及び一般管理費などを扱います。

## 営業利益と経常利益

### 何の問題か

営業利益と経常利益の問題では、増えた費用や収益が「本業の中の話」か「本業以外の話」かを見ます。

### 基本式

| 利益 | 式 | 見るもの |
| --- | --- | --- |
| 営業利益 | 売上高 - 売上原価 - 販売費及び一般管理費 | 本業でもうけた利益 |
| 経常利益 | 営業利益 + 営業外収益 - 営業外費用 | 本業と通常の本業外活動を合わせた利益 |

### 小学生レベルの言い換え

営業利益は「お店の本業でどれくらいもうかったか」です。

経常利益は「本業のもうけに、利息や配当などの本業以外の通常の収入・費用も足し引きしたもの」です。

### 問題文の手がかり

- 営業利益
- 経常利益
- 広告宣伝費
- 販売費及び一般管理費
- 受取配当金
- 営業外収益
- 営業外費用

### 選択肢の切り方

- 広告宣伝費は、販売費及び一般管理費に入る本業の費用なので、営業利益を下げる
- 受取配当金は、営業外収益なので、経常利益に足す
- 本業の費用が増えたら、営業利益から先に下がる
- 営業外収益が増えても、営業利益そのものは増えない

### 計算例

営業利益がもともと90億円で、広告宣伝費が5億円増えた場合、広告宣伝費は販売費及び一般管理費なので営業利益は下がります。

```text
90 - 5 = 85
```

営業外収益が20億円で、受取配当金が3億円増え、営業外費用が16億円なら、経常利益は次のように計算します。

```text
85 + (20 + 3) - 16 = 92
```

この場合は、営業利益が85、経常利益が92です。

## 損益分岐点

### 何の問題か

損益分岐点は、利益がちょうど0になる売上高を求める問題です。

固定費を払うために必要な売上はいくらか、という形で見ます。

### 基本式

```text
限界利益 = 売上高 - 変動費
限界利益率 = 限界利益 ÷ 売上高
損益分岐点売上高 = 固定費 ÷ 限界利益率
```

### 小学生レベルの言い換え

売上が入ったら、まず売れた量に応じて増える費用を払います。

そのあとに残ったお金で、家賃や固定給のような固定費を払います。

```text
売上
-> 変動費を払う
-> 残った割合で固定費を払う
-> 固定費をちょうど払える売上が損益分岐点
```

### 問題文の手がかり

- 売上高
- 変動費
- 固定費
- 材料費
- 外注費
- 製造固定費
- 販売固定費
- 限界利益
- 損益分岐点

### 計算例

売上高が600、材料費240、外注費120、製造固定費90、販売固定費120の場合で考えます。

変動費は、売上に応じて増える費用です。

```text
変動費 = 材料費240 + 外注費120 = 360
```

固定費は、売上が増減しても一定でかかる費用です。

```text
固定費 = 製造固定費90 + 販売固定費120 = 210
```

売上から変動費を引くと、限界利益です。

```text
限界利益 = 600 - 360 = 240
```

売上のうち、固定費を払うために残る割合が限界利益率です。

```text
限界利益率 = 240 ÷ 600 = 0.4
```

固定費210を、売上の40%で払うために必要な売上を出します。

```text
損益分岐点売上高 = 210 ÷ 0.4 = 525
```

この場合、損益分岐点売上高は525万円です。

### 誤答しやすい理由

固定費と変動費を先に分けないまま、売上から費用をまとめて引くと迷いやすくなります。

損益分岐点では、先に変動費を引いて、残る割合で固定費を払うと考えます。

## ROE

### 何の問題か

ROEは、株主が出したお金でどれだけ利益を出したかを見る指標です。

日本語では、自己資本利益率と呼びます。

### 用語の意味

`ROE` は `Return On Equity` の略です。

`Equity` は自己資本、つまり株主のお金として見ます。

### 問題文の手がかり

- 自己資本
- 株主が出したお金
- 株主資本
- 利益率
- ROE

### 似た指標との住み分け

| 用語 | 見るもの | 一瞬フレーズ |
| --- | --- | --- |
| EVA | 企業がどれだけ価値を増やしたか | 経済的付加価値 |
| ROA | 総資産に対する利益 | 全資産で稼ぐ |
| ROE | 自己資本に対する利益 | 株主のお金で稼ぐ |
| ROI | 投資額に対する利益 | 投資で稼ぐ |

### 選択肢の切り方

- 自己資本や株主のお金に対する利益なら、ROE
- 総資産に対する利益なら、ROA
- 投資額に対する利益なら、ROI
- 企業価値の増加なら、EVA

## 流動比率

### 何の問題か

流動比率は、すぐ払う必要がある負債に対して、すぐ使える資産がどれくらいあるかを見る指標です。

短期的に会社が安全か、支払いに困りにくいかを見ます。

### 基本式

```text
流動比率 = 流動資産 ÷ 流動負債
```

### 小学生レベルの言い換え

会社の財布で考えます。

流動資産は、すぐ使えるお金や、すぐ現金化しやすいものです。流動負債は、すぐ返さないといけない借金や支払いです。

財布の中身が、近いうちの支払いより多いかを見るのが流動比率です。

### 問題文の手がかり

- 流動資産
- 流動負債
- 短期的な支払能力
- すぐ使える資産
- すぐ返す負債

### 似た指標との住み分け

| 用語 | 見るもの | 一瞬フレーズ |
| --- | --- | --- |
| 固定比率 | 固定資産と自己資本 | 長く使う設備を自分のお金でまかなえるか |
| 自己資本比率 | 総資本に占める自己資本 | 借金ではないお金の割合 |
| 負債比率 | 自己資本に対する負債 | 借金が自己資本に比べて多いか |
| 流動比率 | 流動資産と流動負債 | すぐ使えるお金 ÷ すぐ返す借金 |

### 選択肢の切り方

- `流動資産` と `流動負債` を比べるなら、流動比率
- 固定資産と自己資本なら、固定比率
- 自己資本の割合なら、自己資本比率
- 負債が自己資本に対してどれくらいかなら、負債比率

## 最後に見る一言

広告費は本業コストで営業利益を下げる。損益分岐点は固定費 ÷ 限界利益率。流動比率は流動資産 ÷ 流動負債。ROEは自己資本でどれだけ稼いだか。

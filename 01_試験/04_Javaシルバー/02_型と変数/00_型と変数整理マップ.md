# 型と変数整理マップ

## 目的

Java Silver の型、変数、代入、キャスト、スコープを整理します。

## 問題で見る手がかり

- プリミティブ型と参照型
- 暗黙の型変換、明示的キャスト
- ローカル変数、フィールド、初期値
- `final`
- スコープ

## まず切る選択肢

- 型が合わない代入
- 初期化前のローカル変数を使っているコード
- スコープ外の変数を参照しているコード
- キャストすればよい場面と、キャストしても危ない場面を混同している選択肢

## 似た構文との住み分け

- フィールドはデフォルト値を持つが、ローカル変数は明示的な初期化が必要
- 小さい型から大きい型へは暗黙変換されやすい
- 大きい型から小さい型へは明示的キャストが必要になりやすい
- 参照型のキャストは、コンパイル可否と実行時例外を分けて見る

## 認知アプローチ

型問題では、先に「代入先の型」を固定します。次に、値の型、変換の可否、スコープ、初期化状態を順番に見ます。

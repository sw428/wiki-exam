# Javaシルバー整理マップ

## 目的

このフォルダは、Java Silver の文法、実行結果、型、例外、APIの安定知識を置く場所です。

## 置くもの

- Java Silver の出題範囲に入る文法整理
- コンパイルエラーと実行時エラーの区別
- 型変換、継承、ポリモーフィズムの判断基準
- コード問題を読む順番
- [`02_検証/04_Javaシルバー`](../../02_検証/04_Javaシルバー/00_Javaシルバー検証マップ.md) から戻した安定知識

## 置かないもの

- 解きかけのコード問題ログ
- 実行結果だけのメモ
- Java Gold寄りの高度な整理

## 処理の流れ

1. コード問題や誤答は、まず検証側に置く
2. 間違えた理由を「構文 / 型 / 継承 / 例外 / API / 実行順」に分ける
3. 次回も使う読み方だけを、このフォルダへ戻す

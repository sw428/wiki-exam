# 基本文法整理マップ

## 目的

Java Silver の基本文法を、コンパイル可否と実行順で判断できる形に整理します。

## 問題で見る手がかり

- `class` / `main` / `static` / `public`
- 演算子、代入、比較
- リテラル、文字列、コメント
- インクリメント、デクリメント

## まず切る選択肢

- 文法として成立しないコード
- 代入と比較を取り違えている選択肢
- 評価順を無視した出力

## 似た構文との住み分け

- `=` は代入、`==` は比較
- `++i` は増やしてから使う、`i++` は使ってから増やす
- `&&` / `||` は短絡評価、`&` / `|` は両辺を評価する

## 認知アプローチ

まず構文が成立するかを見て、次に評価順を追います。出力を考えるのは、コンパイルできると判断してからにします。

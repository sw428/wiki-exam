# データベース整理マップ

## 目的

このフォルダは、応用情報技術者試験のデータベース分野を整理する場所です。

SQL、正規化、トランザクション、ロック、インデックス、設計判断を扱います。

## 問題で見る手がかり

- 表、属性、主キー、外部キー
- SQL、結合、集計、副問合せ
- 正規化、関数従属、冗長性
- トランザクション、排他制御、ロック
- 性能、インデックス

## まず切る選択肢

- 主キーと外部キーを混同している選択肢
- 正規化の目的を取り違えている選択肢
- SQLの結合条件を読み違えている選択肢
- 排他制御と障害回復を混同している選択肢

## 認知アプローチ

データベース問題では、表の形と関係を先に固定します。

```text
表 -> キー -> 関係 -> 制約 -> SQL -> 結果
```

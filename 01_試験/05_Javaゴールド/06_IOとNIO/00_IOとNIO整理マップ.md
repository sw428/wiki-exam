# IOとNIO整理マップ

## 目的

Java Gold の I/O と NIO を、ファイル操作、Path、Files、ストリーム、例外で整理します。

## 問題で見る手がかり

- `Path` / `Paths`
- `Files`
- `InputStream` / `OutputStream`
- `Reader` / `Writer`
- 文字とバイト
- ディレクトリ操作
- 例外

## まず切る選択肢

- 文字ストリームとバイトストリームを混同している選択肢
- `Path` がファイルそのものを表すと誤解している選択肢
- `Files` の戻り値や例外を見落としている選択肢
- 相対パスと絶対パスを混同している選択肢

## 似たAPIとの住み分け

- `Path` はパスを表す
- `Files` はファイル操作を行うユーティリティ
- `InputStream` / `OutputStream` はバイト
- `Reader` / `Writer` は文字

## 認知アプローチ

I/O問題では、対象が「パス」「ファイル操作」「バイト」「文字」のどれかを先に分けます。次に、例外と戻り値を確認します。

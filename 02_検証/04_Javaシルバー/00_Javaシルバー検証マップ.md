# Javaシルバー検証マップ

## 目的

このフォルダは、Java Silver のコード問題、文法判断、実行結果、コンパイルエラーの検証を受ける場所です。

## 置くもの

- 解いたコード問題の要点
- 自分が予想した実行結果
- 実際の正解
- コンパイルエラーと実行時エラーの区別
- 型、継承、例外、APIで迷った点

## 置かないもの

- 安定済みの文法整理
- 実行結果だけの記録
- Java Gold寄りの高度なAPI検証

## 処理の流れ

1. コードのどこから読んだかを書く
2. 実行順、型、例外のどこで判断したかを書く
3. 間違えた場合は、原因を分類する
4. 次回も使える知識は [`01_試験/04_Javaシルバー`](../../01_試験/04_Javaシルバー/00_Javaシルバー整理マップ.md) へ戻す

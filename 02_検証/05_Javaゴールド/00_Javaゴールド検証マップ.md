# Javaゴールド検証マップ

## 目的

このフォルダは、Java Gold のコード問題、API問題、Stream、ラムダ、並行処理などの検証を受ける場所です。

## 置くもの

- 解いたコード問題の要点
- 自分が予想した実行結果
- API仕様の読み違い
- Streamやラムダの評価順
- ジェネリクス、例外、並行処理で迷った点
- Silver知識では足りなかった前提

## 置かないもの

- 安定済みのAPI整理
- API名だけの暗記メモ
- Java Silver側で処理できる基礎文法だけの話

## 処理の流れ

1. コードやAPIのどこを見たかを書く
2. 型、評価順、例外、戻り値のどこで判断したかを書く
3. 間違えた場合は、原因を分類する
4. 次回も使える知識は [`01_試験/05_Javaゴールド`](../../01_試験/05_Javaゴールド/00_Javaゴールド整理マップ.md) へ戻す

API問題では、似たAPI、代入先の型、戻り値、遅延評価、例外発生位置、副作用、スレッド安全性のうち、誤答に関係した観点を選んで確認します。
